##  概要

> **自由で安心な空間を、技術の力で実現する**

**Lumina**(ルミナ)は、監視や検閲、誹謗中傷などが蔓延する現代において、**プライバシー**・**表現の自由**・**心理的安全性**を保障することを目的とした、サイバーセキュリティグループです。

私たちは、技術そのものの設計を通じて、「心理的安全性が確保された空間の創造を通じて、誰もが孤立せず、安心して繋がれる社会の実現」を目指しています。

**法人化について**：Luminaは2026年に法人化予定です。詳細は[概要ページ](https://lumina-group.jp/pages/about.html)をご覧ください。

---

## 私たちの使命

- **デジタル権利の保護**: 基本的人権としてのプライバシーと表現の自由を守ります。
- **ユーザーエンパワーメント**: 技術と教育によって、すべての人が自分らしく力を発揮できる社会を目指します。
- **安全な対話空間**: 誰もが傷つかず、安心して対話できる社会の実現を目指します。
